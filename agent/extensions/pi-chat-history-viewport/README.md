# pi-chat-history-viewport

Minimal extension that only virtualizes the chat history area.

## Commands

- `/chat-viewport on`
- `/chat-viewport off`
- `/chat-viewport toggle`

## Shortcut

- `ctrl+alt+v`

## While enabled

- prompt/editor/footer stay native
- only chat history gets its own viewport
- `PageUp` / `PageDown` scroll chat history
- `Home` jumps to top
- `End` jumps to bottom
